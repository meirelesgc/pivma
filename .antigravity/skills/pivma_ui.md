# PI*VMA UI
Padrões de interface e estilização.

## Ativação
* Modificação ou criação de componentes de interface.

## Regras de Comportamento
* Utilize componentes do Ant Design (v5+).
* Não utilize emotes; utilize ícones do Ant Design.
* Aplique a cor `rgb(0, 156, 59)` através da variável `--primary-color` e do `ConfigProvider`.
* Utilize a fonte 'Inter' para textos de corpo.
* Utilize a fonte 'Instrument Serif' para títulos (H1-H6).
* Utilize propriedades de layout do Ant Design (`Flex`, `Space`, `Row/Col`) em detrimento de CSS.
* Utilize variáveis do arquivo `index.css` para paddings, radius e sombras (ex: `var(--radius-l)`, `var(--shadow-s)`).
* Aplique a classe `.fade-in` para transições de entrada em páginas ou seções.
